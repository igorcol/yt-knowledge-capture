Automated Learning Flow

Automated knowledge extraction pipeline that transforms YouTube videos into structured, strategic Obsidian notes using Whisper V3 (Groq) and Gemini 3 Flash.

Key Features:

High-Fidelity Transcription: Audio extraction and transcription via Groq (Whisper V3) for near-instant results.

AI Synthesis: Applied Pareto Principle to filter 80/20 core concepts from long-form content.

Obsidian Integration: Automated Markdown persistence with metadata for Second Brain workflows.

Frictionless CLI: Designed for speed with global Windows integration.
